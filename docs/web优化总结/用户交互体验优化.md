##### 3. 分页
在实现分页过程中，传统的做法是监听容器的滚动事件，判断是否滚动到容器底部或者离底部一定距离。但这种做法性能较差。

现在可以使用浏览器提供的 `IntersectionObserver` 方法，这个方法性能较好，但有兼容性问题，可以考虑。
